# JS Practice Collection

A small collection of front-end practice projects demonstrating HTML/CSS/JS techniques.

## Projects

- music/
  - index.html — Music player demo
  - css/style.css — Styles for the music demo
  - js/main.js — JavaScript for the music demo
  - music_mp3/ — example audio files
- phase1/
  - index.html — Small interactive page
  - index.css — Styles
  - main.js — Script
- tabUI/
  - index.html — Tabbed UI demo
  - index.css — Styles
  - main.js — Script

## How to run

Open any demo's `index.html` in your browser. For best results run a local static server.

Quick local server (Python 3):

```bash
# from project root
python -m http.server 8000
# then open http://localhost:8000/music/ or /phase1/ or /tabUI/
```

Or use the VS Code Live Server extension.

## Development

- Edit HTML/CSS/JS files in the folders above.
- Use browser DevTools to inspect and debug.

## Notes

- No build tools or dependencies required — static files only.
- Replace or add audio files in `music/music_mp3/` as needed.
